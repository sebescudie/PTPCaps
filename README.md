# PTPCaps
Windows commandline utility to retrieve PTP timestamping capabilities of network adapters using [GetInterfaceSupportedTimestampCapabilities](https://learn.microsoft.com/en-us/windows/win32/api/iphlpapi/nf-iphlpapi-getinterfacesupportedtimestampcapabilities).
